## Features

- **Particle Network Background** — Animated floating dots with connecting lines drawn on a `<canvas>` element
- **Cursor Glow Effect** — A soft radial gradient that follows the mouse cursor
- **Scroll Reveal Animations** — Sections fade up into view using IntersectionObserver
- **Responsive Design** — Adapts cleanly to mobile, tablet, and desktop screens
- **Zero Dependencies** — No npm, no bundler, no build step required
- **Google Fonts** — Uses Syne, DM Mono, and Outfit for a distinctive typographic feel
- **Hover Interactions** — Cards lift and glow on hover throughout the page

## Getting Started

No installation or build tools required.

### 1. Clone the repository

```bash
git clone https://github.com/Sharoonarshad5/portfolio.git
cd portfolio
```

### 2. Open in browser

Simply open `portfolio.html` in any modern browser:

```bash
# macOS
open portfolio.html

# Windows
start portfolio.html

# Linux
xdg-open portfolio.html
```

Or drag and drop the file into your browser window.
